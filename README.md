# Fake Shop

Projeto para treinar cloud e kubernets no desafio devops pro do fabricio veronez, resultados temporariamente disponiveis no ip: http://206.189.254.102

Para rodar esse projeto faça uso de uma ferramenta de k8s rodando um cluster local ou crie uma num serviço de nuvem e usando o kubectl para conectar ao cluster, depois só rodar o kubectl apply no arquivo deployment.yaml

## Variável de Ambiente
DB_HOST	=> Host do banco de dados PostgreSQL.

DB_USER => Nome do usuário do banco de dados PostgreSQL.

DB_PASSWORD	=> Senha do usuário do banco de dados PostgreSQL.

DB_NAME	=>	Nome do banco de dados PostgreSQL.

DB_PORT	=>	Porta de conexão com o banco de dados PostgreSQL.
